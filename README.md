🤖 PostgreSQL Database Agent (Llama 3)
This project is a sophisticated AI Database Agent that uses a local Llama 3 model (via Ollama) to interact with a PostgreSQL database using natural language. The agent can explore tables, understand schemas, and execute SQL queries to answer complex questions.

🚀 Quick Start
To get the project up and running on your local machine, follow these steps:

1. git clone https://github.com/egeyalcin/turkcell-agents
2. cd turkcell-agents
3. chmod +x build.sh
4. ./build.sh
